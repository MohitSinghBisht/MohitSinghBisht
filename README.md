## 👋 About Me

Applied AI Engineer at **Coding Ninjas, Gurgaon**, building production-grade LLM systems optimized for reliability, cost, and traceability.

I focus on:

- **RAG architectures** with vector, keyword, and hybrid retrieval
- **Multi-agent workflows** with LangGraph and LangChain
- **LLM evaluation and observability** with deterministic scoring and human-in-the-loop checks
- **Workflow automation** for internal and customer-facing operations

## 🚀 Featured Work

<table>
<tr>
<td valign="top" width="50%">

### 🎙️ React Mock Interview System
**`LangGraph · LangChain · OpenAI · Whisper · Streamlit`**

- Multi-agent interview simulator with **4 LangGraph agents**: Interviewer, Question Generator, Adaptive Logic, and Evaluator
- Supports **technical** and **coding** interviews with dynamic difficulty progression from easy to hard
- Voice pipeline using **Whisper STT** and **GPT-4o-mini TTS**
- Deterministic scoring with weighted signals such as difficulty, time efficiency, and follow-up penalty
- Final recommendations: **Strong Hire / Hire / No Hire / Strong Reject**

</td>
<td valign="top" width="50%">

### 🎓 Live Class Audit System
**`GPT-4.1 mini · Gemini · Streamlit · Google Sheets`**

- Ingests **Zoom VTT transcripts** and chat exports for automated class review
- Evaluates sessions across **5 parameters** and **14 sub-checks**: behaviour, engagement, content, quality, and escalation
- Designed with strict separation of concerns: **LLM provides qualitative flags, Python owns scoring math**
- Pre-computes response windows to detect missed doubts and student escalation opportunities
- Built for cost efficiency at roughly **₹2 per audit**

</td>
</tr>
<tr>
<td valign="top" width="50%">

### 📄 AI Resume Analyzer
**`Flask · React · GPT-4.1 mini · Netlify · Render`**

- Full-stack resume analysis tool benchmarked against **300+ Indian DA job listings** across **3 experience tiers**
- Sequential workflow: **gap analysis → ATS score → strategy → rewrite → tracking**
- Anti-hallucination guardrails include **verbatim title extraction**, section boundaries, and date-anchored experience calculation
- Deployed with **Netlify** for frontend and **Render** for backend

</td>
<td valign="top" width="50%">

### 🤖 Sales Counsellor Bot
**`LangChain · Qdrant · HNSW · OpenAI · Streamlit`**

- RAG system built on **Qdrant** with **HNSW indexing** for fast semantic retrieval
- Dual-LLM architecture separates **query understanding** from **response generation**
- Uses **score-based retrieval** with similarity thresholds for better answer precision
- Supports **English + Hinglish** with automatic language detection
- Auto-extracts leads into CSV for CRM handoff
